杭州市第一人民医院简介

杭州市第一人民医院创建于1923年5月29日，是杭州地区融医疗、教学、科研、预防和社会保健于一体的市属最大的综合性三级甲等医院，浙江省医学科学院临床研究院、南京医科大学附属杭州医院。作为杭州地区骨干医疗机构，市一院也是杭州市危重孕产妇抢救中心，杭州市药剂、放射、病理、护理、急救、肿瘤、检验、产科、院感、骨科、ICU等质控中心的挂靠单位。

医院现有职工1903人，其中中高级职称卫技人员超过50%，享受国务院和市政府特殊津贴11人，省市各级新世纪培养人才48人，杭州市钱江特聘专家3人，南京医科大学教授、副教授30人，博导、硕导42人，浙江大学医学院兼职教授、副教授23名。

院区占地30余亩，核定床位1000张，实际开放床位1300张。现有一、二级临床医疗和医技科室48个，其中消化内科、心血管内科、眼科、妇科、神经内科、肾内科、血液内科、临床检验学、耳鼻咽喉科、小儿内科、肿瘤放疗学、产科（危重症）为省市级医学重点学科。

作为南京医科大学附属医院，浙江大学医学院、杭州师范大学、浙江中医药大学、温州医学院等大专院校的教学医院，医院每年承担400余名医学生的临床实习教学工作，同时设有浙江中医药大学博士教学点、南京医科大学与温州医学院硕士教学点，医学理论教学体系较为完整。

因一以贯之的优质医疗水平、先进的硬件设施和以人为本的服务理念，医院在杭州地区乃至浙江省享有很好的美誉度和覆盖率，先后荣获了全国卫生系统先进集体、全国百姓放心示范医院、浙江省文明单位、杭州市文明单位、浙江省文明示范医院、卫生部院务公开示范医院等荣誉。“市一人”将始终铭记“敬业、严谨、诚信、仁爱”之院训，全心全意地为广大患者服务。

随着医疗卫生体制改革的逐步推进，在市委、市政府统一部署，上级卫生行政主管部门领导下，市一医院以多种形式与数家医疗机构或单位合并、合作，稳步踏上医疗集团化发展新历程：

2000年9月，杭州玻璃集团公司职工医院成建制划拨；2008年1月，与杭州市妇幼保健院实行一体化管理；2009年5月，与杭州市钱江新城管委会签署“杭州市妇女医院”合作建设协议；2009年12月，收购杭州第四医院65%股权。

市一医疗集团的建设，是以市一医院为核心，对五院现有医疗资源的有效整合与合理配置，从而发挥名院品牌效应，提升新建院区发展平台，降低运营风险，实现整个医疗集团的可持续发展。

目前各院区错位发展，各具特色。杭州市第一人民医院：承担集团内业务指导、疑难危重病症诊治、医学科研和教学等任务，提供人才、资金、技术、设备、经营协作等方面支持。杭州市妇幼保健院: 全杭州地区妇幼保健业务的指导中心。杭州市肿瘤医院（吴山院区）:逐步建成集医疗、保健、预防、康复为一体的三级肿瘤专科医院。杭州市妇女医院（钱江新城院区）：集医疗、预防、科研、保健于一体的国内一流，省内领先的三级甲等专科医院，2013年建成开业。杭州市老年病医院（城北院区）：以老年和卒中康复、呼吸支持、腹膜透析、褥疮治疗、临终关怀等重点学科为特色的老年病专科医院。
